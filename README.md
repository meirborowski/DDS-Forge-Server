## 🛠️ Getting Started

Ready to fire up this beast? Let's go! 🚀

### 📦 Install Dependencies

```bash
npm install
```

### 🎮 Run the Server

```bash
# 🔧 Development mode (for tinkering)
npm run start

# 👀 Watch mode (auto-reload magic)
npm run start:dev

# 🚀 Production mode (full power!)
npm run start:prod
```

### 🧪 Testing Suite

Because we believe in quality! ✨

```bash
# 🔬 Unit tests
npm run test

# 🌐 End-to-end tests
npm run test:e2e

# 📊 Test coverage
npm run test:cov
```
